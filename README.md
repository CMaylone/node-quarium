Rasberry Pi Aquarium Monitor Built On Node :fish:

# Installation

## Install Node.js on Rasberry Pi

```shell
sudo apt-get update
sudo apt-get upgrade
sudo wget http://node-arm.herokuapp.com/node_latest_armhf.deb
sudo dpkg -i node_latest_armhf.deb
```
